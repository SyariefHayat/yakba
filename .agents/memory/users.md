# Persona Pengguna

## 1) Orang Tua (Primer)
- Tujuan: mencari bimbel terpercaya dan produk edukasi untuk anak.
- Kebutuhan utama: informasi manfaat Yakba, detail program, kurikulum, harga produk, CTA cepat ke WhatsApp.
- Perilaku: dominan akses mobile, butuh keputusan cepat berbasis trust.

## 2) Calon Guru/Mitra (Sekunder)
- Tujuan: memahami model kemitraan dan peluang membuka pusat bimbingan.
- Kebutuhan utama: penjelasan skema kemitraan, syarat dasar, jalur kontak jelas.

## 3) Tim Internal Yakba (Admin/Editor)
- Tujuan: memperbarui konten produk/program tanpa perubahan kode.
- Kebutuhan utama: alur edit cepat, aman, dan terstruktur.
