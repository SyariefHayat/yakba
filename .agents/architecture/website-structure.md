# Struktur Website & Informasi

## Halaman Inti MVP
- Beranda (Landing): value proposition, trust elements, CTA utama.
- Tentang Yakba: profil, misi, diferensiasi.
- Program Bimbingan: overview program, detail kurikulum, hasil belajar.
- Katalog Produk: daftar produk Digital/Fisik dengan filter kategori.
- Detail Produk: informasi lengkap + CTA WhatsApp terprefill.
- Partnership Guru/Mitra: penjelasan model kemitraan + CTA WhatsApp.
- Kontak: jalur komunikasi resmi.

## Struktur Navigasi Disarankan
- Menu utama: Beranda, Program, Produk, Kemitraan, Tentang, Kontak.
- CTA global: tombol WhatsApp pada navbar/floating button.

## Prioritas IA
- Prioritas 1: jalur orang tua dari landing -> program/produk -> WhatsApp.
- Prioritas 2: jalur mitra dari landing -> kemitraan -> WhatsApp.
