# Integrasi Eksternal

## 1) WhatsApp
Tujuan: menjadi jalur utama transaksi dan inquiry.

Kebutuhan:
- Deep link WhatsApp di halaman produk/program.
- Prefilled message menyertakan konteks item.
- Tracking event klik CTA per halaman dan per item.

## 2) Analytics
Tujuan: mengukur performa funnel dan KPI PRD.

Event minimum:
- `page_view`
- `product_view`
- `program_view`
- `cta_whatsapp_click`

Metrik utama:
- Monthly unique visitors.
- CTR WhatsApp.
- Konversi transaksi bulanan (diinput manual dari operasional WhatsApp).

## 3) CMS/Admin (internal integration)
Tujuan: update konten tanpa deploy.

Kebutuhan:
- CRUD produk.
- CRUD informasi program.
- Manajemen status publish.
