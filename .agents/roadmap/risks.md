# Risiko Produk & Teknis

## Risiko Operasional
- Lonjakan chat WhatsApp dapat menjadi bottleneck karena proses transaksi manual.
- Data transaksi bisa tidak konsisten jika pencatatan manual tidak disiplin.

## Risiko Produk
- Brand guideline belum lengkap dapat memicu inkonsistensi visual dan pesan.
- Scope creep dari visi jangka panjang berpotensi mengganggu target MVP 6-8 minggu.

## Risiko Teknis
- Kualitas pelacakan funnel rendah jika event analytics tidak didefinisikan sejak awal.
- Konten cepat usang jika workflow CMS/admin tidak sederhana.

## Mitigasi Awal
- Definisikan template pesan WhatsApp dan SOP operasional transaksi.
- Lock scope MVP melalui daftar non-goals yang eksplisit.
- Tetapkan event analytics wajib sebelum go-live.
- Tunjuk owner konten internal dengan SLA update konten.
